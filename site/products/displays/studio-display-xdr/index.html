Studio Display XDR

Studio Display XDR. Built for pros.

A pro reference display with Mini-LED XDR backlighting, ProMotion, and reference modes for color-critical workflows. Designed for the highest-end Mac configurations and the most demanding workflows.

Starting at $3,999

At a glance

Pro display. Pro workflow.

Key features

What makes it special.

  • Mini-LED XDR backlighting

    Thousands of mini-LEDs provide extreme contrast and brightness — true reference-grade HDR.

  • 6K resolution

    20.4 million pixels for incredible detail in pro photo, video, and 3D workflows.

  • ProMotion up to 120Hz

    Fluid scrolling and responsive interaction; benefits creative apps with high frame-rate previews.

  • Reference modes

    Built-in reference modes for color-critical work — and the ability to define custom modes.

  • Nano-texture option

    For studios with significant glare, the nano-texture glass scatters light to maintain image contrast.

  • Pro Stand

    Optional Pro Stand offers counterbalanced tilt, height, and rotation. VESA mount adapter sold separately.
